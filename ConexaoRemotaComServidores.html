Conexão de Área de Trabalho Remota (Windows)

Através desta conexão, é possível conectar-se à outros servidores (máquinas), seja ela local ou via cloud. Sendo que para isso é necessário que ambom compartilhem uma mesma conexão de rede. Dessa forma, é possível acessar os programas, arquivos e recursos de rede de um outro computador.

Para isso, o computador a ser hospedado deve estar ligado, ter uma conexão de rede, a área de trabalho remota deve estar habilitaa, é preciso ter acesso de rede ao servidor remoto e ter a permissão de conexão.

OBS: É preciso estar na lsia de usuários para tal permissão.

OBS: É recomendável pesquisar o nome do servidor a se hospedado e se as conexões de área de trabalho são permitidas pleo firewall.

  • Permissão de conex~es remotas no servidor a ser hospedado

    1. Vá nas propriedades do computador, em configurações remotas e em Área de trabalho remota;
    2. Vá em Selecionar usuários e adicione sua conta de usuário.

  • Para permitir conexões de área de trabalho por firewall

    • Essas conexões podem ser bloqueadas por meio de um firewall, sendo preciso mudar suas configurações para isso.

    1. Vá em Painel de controle e clique em ,Sistema e segurança;
    2. Clique em Permitir um programa pelo firewall e Alterar configurações e então marque a opção Área de trabalho remota.

    OBS: Lembre-se que sua conta de usuário deve ter uma senha para o possível acesso a área de trabalho remota para conexão local.

  • Iniciar a conexão

    • Na aba de pesquisa do windows, digite o nome mstsc para exibir o software de conexão remota.

    1. Indique o IP do servidor a ser hospedado;
    2. Na opção Recursos Locais é possível escolher o que será compartilhado do servidor local para o servidor a ser hospedado, através da opção mais em dispositivos.
